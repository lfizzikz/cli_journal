# How to use:

## Commands
- search
    - -date 
        - {YYYY-MM-DD}
    - -year
        - {YYYY}
    - -month
        - {MM}
    - -from
        - {YYYY-MM-DD}
    - -to
        - {YYYY-MM-DD}
- add
    - -tag
    - two ways to use add
        - add {stuff you want to write}
        - add {press enter and will open a stdout with multi-line support.}
            - press "ctrl-d" to exit once done
- open
    - today
        - opens todays file in editor of choice
- config
    - -editor
        - nvim
        - obsidian
